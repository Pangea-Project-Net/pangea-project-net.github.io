## Release Notes

### Highlights

- **Pages editor polish**: CTRL‑S save, CTRL‑F in‑page search, spell checking with suggestions, list paste improvements, and right‑click selection fixes.
- **Communities improvements**: delete/archive/unarchive workflows, role‑based permissions enforcement, community sharing enhancements, and persistence fixes.
- **Security hardening (P0 fixes)**: upgraded to AES‑256‑GCM for private key encryption, PBKDF2‑HMAC‑SHA256 password hashing, cryptographic salts, and removal of credential leaks.

### Known Issues

- User settings UI mixes device- and user-scoped settings (Bug #004).
- Admin user list timestamps may display incorrect dates.
- NAT traversal can be inconsistent in complex network environments.
- Streaming is not yet fully optimized (paused pending architecture improvements).
